Inside icon

Realization of decentralized
governance

ICON은 ICON Nexus, ICON Senate,
ICON Mint로 이루어져 있습니다

  1. ICON Nexus는 다양한 블록체인을 연결합니다
  2. ICON Senate은 운영·정책 합의 기구입니다
  3. ICON Mint는 통화량을 안정적으로 조절합니다

ICON Nexus

ICON Nexus를 통해 블록체인 사이의 빠르고 신뢰성 있는
가치이동을 가능하게 합니다
상이한 운영정책을 가진 ICON Nexus들을 연결할 수 있어
확장성이 뛰어납니다
ICON Nexus는 이미 금융기관에서 사용 중인 loopchain을
바탕으로 작동하며, 개선된 BFT(Byzantine Fault Tolerance) 알고리즘을 기반으로 실시간 거래를 지원합니다

ICON Senate

ICON Senate는 네트워크 참여자들로 구성된 주요 운영
정책을 제정·관리하는 기구로 탈중앙화된 거버넌스를
실현합니다

ICON Mint

ICON Mint는 AI예측모델을 활용하여 네트워크 내 가상화폐
교환비율 및 통화량을 안정적으로 관리합니다
