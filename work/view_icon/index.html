Inside icon

Realization of decentralized
governance

ICON consists of ICON Nexus, ICON Senate,
and ICON Mint :

  1. ICON Nexus connects various blockchains
  2. ICON Senate is a consensus organization of
    participants’ operational policy
  3. ICON Mint controls the money supply stably

ICON Nexus

Various blockchains can move value quickly and reliably
through ICON Nexus
ICON Nexus supports real-time trading based on
improved Byzantine Fault Tolerance (BFT) consisting of
loopchain in use by financial institutions
An ICON Nexus can be extended with connectivity to others with different governances

ICON Senate

ICON Senate provides decentralized governance by
establishing and managing operational policies with
agreements among network participants

ICON Mint

ICON Mint manages virtual currency exchange rates and
money supply reliably within the network by using the AI
predication model

the loop
